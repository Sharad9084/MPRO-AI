# MPRO Auditor Accounting

Working local MVP for MPRO invoice accounting and Auditor reconciliation.

## Open

Open `index.html` in a browser.

Auditor login:

- Username: `auditor@mpro.com`
- Password: `Auditor@2026`

## Current Flow

- Launch screen
- Role selection with 4 roles
- Auditor Login active for now
- Dashboard navbar after login
- Global filters from uploaded Excel/CSV data
- Accounting filters from uploaded Excel/CSV data
- Missing filter columns stay visible but disabled until that column exists
- Editable, sortable, resizable, reorderable invoice table
- Browser local save and CSV export
